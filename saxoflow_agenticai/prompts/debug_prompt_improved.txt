2. FIND ROOT CAUSE
   - Quote exact error from log
   - Map to issue category (RTL, TB, environment)
   - Check if error repeats or is one-time

3. MAP TO AGENT
   - RTL syntax/width/port/logic errors → RTLGenAgent

   - Verilog-2001 only (no SystemVerilog)
