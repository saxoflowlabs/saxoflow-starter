# Getting Started Example

This small counter project gives new SaxoFlow users a clean HDL file layout to inspect.

Files:

- `source/specification/counter.md`: short design intent.
- `source/rtl/verilog/counter.v`: synthesizable Verilog counter.
- `source/tb/verilog/counter_tb.v`: Icarus-compatible testbench.

Create your own project with:

```bash
saxoflow unit projects/my_counter
```
